WHAT IS JAVASCRIPT ?

JavaScript is a scripting or programming language that allows you to implement complex features on web pages. It is one of the core technologies of the World Wide Web, alongside HTML and CSS1. JavaScript is used to create interactive effects within web browsers2. It is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions3. While it is most well-known as the scripting language for Web pages, many non-browser environments also use it, such as Node.js, Apache CouchDB and Adobe Acrobat3.
